# StudyMate-AI
StudyMate Timer – A simple Pomodoro focus timer with an animated progress ring, built using HTML, CSS, and JavaScript to help users stay productive during study sessions.
It helps users stay productive by breaking work into focused time sessions with short breaks. 
The timer includes an animated progress ring and start/pause controls for a smooth study experience.
Features
- Pomodoro focus timer
- Animated circular progress indicator
- Start and pause controls
- 25 / 5 / 15 minute session options
- Clean and responsive UI
